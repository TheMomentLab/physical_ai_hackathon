### 📝 1종목 미션 (Task 1 Mission)
> **"한쪽 팔로 공을 집어 반대쪽 팔의 그리퍼로 전달한 뒤, 색상별로 수납함에 정확히 분류"**

- **주요 동작**:
  1. **Pick & Place**: 한쪽 팔을 이용해 테이블 위의 공을 집습니다.
  2. **Transfer**: 집은 공을 반대편 팔의 그리퍼로 안정적으로 전달합니다.
  3. **Classification**: 전달받은 공의 색상을 인식하여 지정된 수납함에 정확히 분류합니다.
